About This Project

This IRCTC Ticket Booking clone is a full-stack web application inspired by real-world train booking systems. The backend is built in Java using Servlets and JDBC to process user actions such as searching, booking, and canceling tickets. The project demonstrates end-to-end workflow integration between the frontend and backend.
